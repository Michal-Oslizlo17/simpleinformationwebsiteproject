Obecny temat to: CSS


Kilka praktycznych przykladów


---

Zmiana koloru tła strony:
---

body { background-color: #f2f2f2; }
---

Dodanie stylu dla nagłówka h1:
---

h1 { font-size: 36px; color: #333; text-align: center; margin-top: 50px; }
---

Zmiana koloru tła i koloru czcionki dla przycisków:
---

button { background-color: #4CAF50; color: white; padding: 10px 20px; border: none; border-radius: 4px; cursor: pointer; }
button:hover { background-color: #3e8e41; }
---

Ustawienie obrazka tła na stronie:
---

body { background-image: url('background-image.jpg'); background-repeat: no-repeat; background-size: cover; }
---

Dodanie stylu dla tabeli:
---

table { border-collapse: collapse; width: 100%; }
th, td { text-align: left; padding: 8px; }
th { background-color: #4CAF50; color: white; }
tr:nth-child(even) { background-color: #f2f2f2; }
---

Zmiana stylu linków:
---

a { color: #4CAF50; text-decoration: none; }
a:hover { text-decoration: underline; }
---

Dodanie animacji dla przycisku:
---

.button { background-color: #4CAF50; color: white; padding: 10px 20px; border: none; border-radius: 4px; cursor: pointer; transition-duration: 0.4s; }
.button:hover { background-color: #3e8e41; }
---
