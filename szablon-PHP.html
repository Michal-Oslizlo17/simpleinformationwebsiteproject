Obecny temat to: JavaScript


<?php
$servername = "localhost";
$username = "nazwa_uzytkownika";
$password = "haslo";
$dbname = "nazwa_bazy_danych";

$conn = mysqli_connect($servername, $username, $password, $dbname);

if (!$conn) {
die("Połączenie nieudane: " . mysqli_connect_error());
}
echo "Połączenie udane";
?>



<?php
$sql = "INSERT INTO tabela (kolumna1, kolumna2, kolumna3)
VALUES ('wartosc1', 'wartosc2', 'wartosc3')";

if (mysqli_query($conn, $sql)) {
echo "Nowy rekord został dodany";
} else {
echo "Błąd: " . $sql . "<br>" . mysqli_error($conn);
}
?>



<?php
$sql = "SELECT kolumna1, kolumna2, kolumna3 FROM tabela";
$result = mysqli_query($conn, $sql);

if (mysqli_num_rows($result) > 0) {
while($row = mysqli_fetch_assoc($result)) {
echo "kolumna1: " . $row["kolumna1"]. " - kolumna2: " . $row["kolumna2"]. " - kolumna3: " . $row["kolumna3"]. "<br>>";
}
} else {
echo "Brak wyników";
}

mysqli_close($conn);
?>



Dlaczego wykorzystujemy PHP i podaj praktyczne przykłady:
PHP jest językiem skryptowym, który jest często wykorzystywany do tworzenia stron internetowych i aplikacji internetowych. Wykorzystuje się go do tworzenia dynamicznych stron internetowych, czyli takich, które zawierają treści, które mogą się zmieniać w zależności od czasu lub użytkownika. Przykłady zastosowań PHP to:
Formularze kontaktowe: PHP może być wykorzystywany do tworzenia formularzy kontaktowych na stronach internetowych, które umożliwiają użytkownikom przesłanie wiadomości bezpośrednio do właściciela strony.
Systemy logowania: PHP może być wykorzystywany do tworzenia systemów logowania na stronach internetowych, które umożliwiają użytkownikom logowanie się na swoje konta i dostęp do zasobów zabezpieczonych hasłem.
Sklepy internetowe: PHP może być wykorzystywany do tworzenia sklepów internetowych, które umożliwiają użytkownikom zakupy online i przetwarzanie płatności.
Blogi: PHP może być wykorzystywany do tworzenia blogów, które umożliwiają użytkownikom publikowanie treści, komentowanie wpisów i udostępnianie treści w mediach społecznościowych.
Portale społecznościowe: PHP może być wykorzystywany do tworzenia portali społeczności
